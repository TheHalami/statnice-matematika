# Státnice z matematiky

Statický web pro GitHub Pages.

## Nasazení přes GitHub
1. Vytvoř public repozitář, například `statnice-matematika`.
2. Nahraj soubor `index.html` do rootu repozitáře.
3. V `Settings → Pages` nastav:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
4. Počkej pár minut a otevři URL ve tvaru:
   `https://TVUJ_USERNAME.github.io/statnice-matematika/`
